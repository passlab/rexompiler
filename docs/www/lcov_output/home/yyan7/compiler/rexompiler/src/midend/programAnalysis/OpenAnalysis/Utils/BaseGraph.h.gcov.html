      63             : /** BaseGraph is the abstract base class (the "interface") for a general graph.  It defines graph properties common to
      64             :     directed and undirected graphs.  It leaves out some loose ends in the interface:
      65             :     1. A node has no notion of incident edges or neighboring nodes because the number of kinds of incident edges or
      66             :        neighboring nodes is dependent upon the graph being directed or undirected.
      67             :     2. For the same reason, the method delete(node) cannot delete any incident edges.  Therefore the directed or
      68             :        undirected graph *must* override it with a more complete version.  Similarly, the method delete(edge) cannot
      69             :        delete the edge from the list(s) of the nodes involved.
      70             :     3. In an analogous manner, the method add(edge) must be overridden with a more complete version that adds the edge
      71             :        into the records of the nodes involved, if needed.
      72             : 
      73             :     The only restriction on nodes and edges is that they should be unique objects, meaning, an edge or a node cannot
      74             :     be shared between two graphs.  A node or edge can also not be inserted twice.  Nodes and edges are identified by
      75             :     their pointer values (BaseGraph::Node* and BaseGraph::Edge*).

      77             :     Following exceptions are thrown by the class (all are subclasses of Exception):
