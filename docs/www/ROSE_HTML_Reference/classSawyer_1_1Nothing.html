Sawyer::Nothing Class Reference

Description

Represents no value.

An object of type Nothing can be assigned to any Optional object to make it contain nothing.

Optional<int> opt1;
Optional<int> opt2(Nothing());
Optional<int> opt3 = 123;
opt3 = Nothing();
