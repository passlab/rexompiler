template<class I>
class Sawyer::Container::IntervalSet< I >::ConstScalarIterator

Scalar value iterator.

Scalar value iterators iterate over each scalar value in the set. Two caveats to beware of:

  • The set can hold a very large number of values, even the entire value space, in which case iterating over values rather than storage nodes could take a very long time.
  • Iterating over values for a non-integral type is most likely nonsensical.
