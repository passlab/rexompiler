Sawyer::Tree::ConsistencyException Class Reference

Description

Exception if tree consistency would be violated.

If the user attempts to modify the connectivity of a tree in a way that would cause it to become inconsistent, then this exception is thrown. Examples of inconsistency are:

  • Attempting to insert a node in such a way that it would have two different parents.
  • Attempting to insert a node at a position that would cause it to be a sibling of itself.
  • Attempting to insert a node in such a way that it would become its own parent.
  • Attempting to insert a node in such a way as to cause a cycle in the connectivity.
