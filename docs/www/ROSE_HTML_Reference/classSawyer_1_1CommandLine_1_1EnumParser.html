Sawyer::CommandLine::EnumParser< T > Class Template Reference

Description

template<typename T>
class Sawyer::CommandLine::EnumParser< T >

Parses an enumerated constant.

Parses one of the strings added via with and returns the associated enumerated constant. The template parameter T is the enum type. Returns A ParsedValue whose value if type T. This parser uses StringSetParser, thus it prefers to match enum members with the longest names.
